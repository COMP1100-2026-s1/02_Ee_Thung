## Summary: Evidence-to-Feature Traceability

| MVP Feature | Core Barrier Addressed | Key Evidence |
|---|---|---|
| Live Discovery Feed | Information fragmentation | LC1 (80% miss events), LC2 (90% faster on live feed) |
| Skill-Tiered Open Socials | Social anxiety / fear of inadequacy | LC4 (80% fear attending alone), LC1 MVP (8/10 chose tiered mockup) |
| Pathfinder Booking | Isolation / hesitation to engage | LC8 (formal booking fails), LC9 (65% would use casual chat) |
| Clubs Directory | Decision-making friction | LC6 (75% prefer category browsing), LC4 MVP (85% trust with price + memories) |
| In-App Messaging | Information fragmentation / isolation | LC1 (80% miss events due to scattered comms), LC7 (90% want more human connection pre-join) |
